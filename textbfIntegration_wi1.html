Integration with AWS Elemental MediaLive
AWS Elemental MediaPackage seamlessly integrates with AWS Elemental MediaLive, the live video encoding service. This integration allows users to create end-to-end workflows for live streaming, from content origination and encoding to packaging and delivery, all within the AWS environment 11AWS Elemental.
